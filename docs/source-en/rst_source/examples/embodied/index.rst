Embodied Scenarios
================================

This category includes embodied training examples with SOTA models (e.g., OpenVLA, OpenVLA-OFT, GR00T, Dexbotic) and different benchmarks (e.g., LIBERO, ManiSkill),
as well as reinforcement learning training examples on real robots.
